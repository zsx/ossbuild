tcpserversink — Send data as a server over the network via TCP

Synopsis

Description

Example launch line

# server:
gst-launch fdsrc fd=1 ! tcpserversink protocol=none port=3000
# client:
gst-launch tcpclientsrc protocol=none port=3000 ! fdsink fd=2
