volume

volume — Set volume on audio/raw streams

Synopsis

                    GstVolume;

Object Hierarchy

  GObject
   +----GstObject
         +----GstElement
               +----GstBaseTransform
                     +----GstAudioFilter
                           +----GstVolume

Implemented Interfaces

GstVolume implements GstImplementsInterface and GstMixer.

Properties

  "mute"                     gboolean              : Read / Write
  "volume"                   gdouble               : Read / Write

Description

The volume element changes the volume of the audio data.

Example launch line

gst-launch -v -m audiotestsrc ! volume volume=0.5 ! level ! fakesink silent=TRUE
This pipeline shows that the level of audiotestsrc has been halved (peak values are around -6 dB and RMS around -9 dB) compared to the same pipeline without the volume element.
