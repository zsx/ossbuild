gconfaudiosink

gconfaudiosink — Audio sink embedding the GConf-settings for audio output

Synopsis

Implemented Interfaces

GstGConfAudioSink implements GstChildProxy.

Properties

  "profile"                  GstGConfProfile       : Read / Write

Description

This element outputs sound to the audiosink that has been configured in GConf by the user.

Example launch line

gst-launch filesrc location=foo.ogg ! decodebin ! audioconvert ! audioresample ! gconfaudiosink
Play on configured audiosink
